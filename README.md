# 🌒 Twilight Gallery

**Twilight Gallery** is a dark-themed art gallery web project focused on immersive design and storytelling.

The website allows visitors to explore mysterious paintings, read their lore, and access the **Marketplace of Shadows** where artworks can be viewed and purchased.

This project is being developed gradually and will continue to expand with additional features such as auctions, user accounts, and a richer gallery experience.

---

## ✨ Current Features

- 🏛️ **Home Page**
  - Introduction to the Twilight Gallery
  - Atmospheric design and visuals

- 🛒 **Marketplace**
  - Browse paintings available for purchase
  - Entry point to the Marketplace of Shadows

- 🧾 **Marketplace Registration**
  - Registration page for users who wish to access marketplace features

- 🌌 **Lore & Atmosphere**
  - Story-driven presentation of artworks and gallery theme

---

## 🌌 Project Vision

Twilight Gallery aims to create an immersive digital gallery where each artwork feels like a piece of a larger story.

The long-term vision is to build a platform where users can:

- Discover mysterious paintings
- Explore their lore and origins
- Purchase artworks through a marketplace
- Participate in auctions for rare pieces

The goal is to blend **art, storytelling, and web design** into a unique gallery experience.
